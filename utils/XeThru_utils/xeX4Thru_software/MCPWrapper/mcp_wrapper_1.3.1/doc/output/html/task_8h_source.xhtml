1177 /*
1178  * THIS FUNCTION MUST NOT BE USED FROM APPLICATION CODE. IT IS AN
1179  * INTERFACE WHICH IS FOR THE EXCLUSIVE USE OF THE SCHEDULER.
1180  *
1181  * THIS FUNCTION MUST BE CALLED WITH INTERRUPTS DISABLED.
1182  *
1183  * Removes the calling task from the ready list and places it both
1184  * on the list of tasks waiting for a particular event, and the
1185  * list of delayed tasks. The task will be removed from both lists
1186  * and replaced on the ready list should either the event occur (and
1187  * there be no higher priority tasks waiting on the same event) or
1188  * the delay period expires.
1189  *
1190  * @param pxEventList The list containing tasks that are blocked waiting
1191  * for the event to occur.
1192  *
1193  * @param xTicksToWait The maximum amount of time that the task should wait
1194  * for the event to occur. This is specified in kernel ticks,the constant
1195  * portTICK_RATE_MS can be used to convert kernel ticks into a real time
1196  * period.
1197  */
