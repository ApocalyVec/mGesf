283 /*
284  * xSemaphoreAltTake() is an alternative version of xSemaphoreTake().
285  *
286  * The source code that implements the alternative (Alt) API is much
287  * simpler because it executes everything from within a critical section.
288  * This is the approach taken by many other RTOSes, but FreeRTOS.org has the
289  * preferred fully featured API too. The fully featured API has more
290  * complex code that takes longer to execute, but makes much less use of
291  * critical sections. Therefore the alternative API sacrifices interrupt
292  * responsiveness to gain execution speed, whereas the fully featured API
293  * sacrifices execution speed to ensure better interrupt responsiveness.
294  */
