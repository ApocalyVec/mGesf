Create a new RegMap object.

The RegMap class is intended to be sub-classed for a specific register map on a specific chip. One register map belongs to one register block. The subclass must define or assign the set_register and get_register to a function that knows how to access the physical chip registers.

Parameters
blockA pointer to the parent class
nameThe name of this object.
