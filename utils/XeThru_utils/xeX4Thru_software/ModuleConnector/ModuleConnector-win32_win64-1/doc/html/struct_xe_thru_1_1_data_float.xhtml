Detailed Description

Encapulates a vector of float elements, for example baseband data.

This package can be retrieved from the module with the use of read_message_data_float in the XEP interface.

Python warning: Accessing vectors directly can cause memory corruption if the parent object goes out of scope and is garbage collected. Use accessor methods for a workaround.
