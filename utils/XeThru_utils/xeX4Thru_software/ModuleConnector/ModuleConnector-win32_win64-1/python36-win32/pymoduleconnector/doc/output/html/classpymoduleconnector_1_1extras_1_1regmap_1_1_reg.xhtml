Detailed Description

Generic register class.

This class is intended to be sub-classed for each specific register. The sub-classing enables us to use class attributes for each register segment without the different registers interfering with each other.

The register maintains a shadow copy of the last known value. This should be used with care since it could easily become out of sync with the chip, for instance in status registers.
