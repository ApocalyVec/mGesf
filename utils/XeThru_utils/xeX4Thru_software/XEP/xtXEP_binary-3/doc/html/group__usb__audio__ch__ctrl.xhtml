Detailed Description

This section lists the available controls for each channel of the audio interface.

Each channel can have any control combination; simply perform a bitwise OR ('|') to combine several values.

  1. AUDFeatureUnitDescriptor_MUTE
  2. AUDFeatureUnitDescriptor_VOLUME
  3. AUDFeatureUnitDescriptor_BASS
  4. AUDFeatureUnitDescriptor_MID
  5. AUDFeatureUnitDescriptor_TREBLE
