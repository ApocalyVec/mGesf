"SAM E70 Xplained - PIO definitions"

This pages lists all the PIOs definitions contained in board.h.

The constants are named using the following convention: PIN_* for a constant which defines a single Pin instance (but may include several PIOs sharing the same controller), and PINS_* for a list of Pin instances.
