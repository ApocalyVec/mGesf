Detailed Description

Definitions and classes for USB Audio class descriptors.

  • For a USB device:
    1. When declaring an Audio-specific descriptor, use the descriptor types and subtypes defined in this unit (see "USB Audio descriptor types" and "USB Audio descriptor subtypes").
    2. When declaring the device descriptor of a USB audio device, use "USB Audio device descriptor values" defined here.
    3. Declare an AUDStreamingInterfaceDescriptor instance as part of the configuration descriptors returned by a USB audio device.
    4. Declare a AUDHeaderDescriptor as part of the configuration descriptors of the device.
    5. Declare an AUDFeatureUnitDescriptor instance as part of the configuration descriptors returned by a USB audio device.
    6. Declare an instance of AUDFormatTypeOneDescriptor as part of the configuration descriptors of an audio device.
