§ AUDFeatureUnitDescriptor

AUDFeatureUnitDescriptor

Describes available controls for each channel of the unit or terminal it is connected to.

This type must be augmented with the relevant number of bmaControls fields and the iFeature field.

§ AUDFormatTypeOneDescriptor

AUDFormatTypeOneDescriptor

Describes an audio data stream that is constructed on a sample-by-sample basis.

This type must be augmented with either the continuous sampling frequency range (if bSamFreqType = <AUDFormatTypeOneDescriptor_CONTINUOUS>) or with an array containing the possible discrete frequencies.

§ AUDHeaderDescriptor

AUDHeaderDescriptor
