§ MCI_SetSpeed()

static uint32_t MCI_SetSpeed ( sMcid *  pMci,
uint32_t  mciSpeed,
uint32_t  mck 
)
static

Configure the MCI CLKDIV in the MCI_MR register.

The max. for MCI clock is MCK/2 and corresponds to CLKDIV = 0

Parameters
pMciPointer to the low level MCI driver.
mciSpeedMCI clock speed in Hz, 0 will not change current speed.
mckMCK to generate MCI Clock, in Hz
Returns
The actual speed used, 0 for fail.
